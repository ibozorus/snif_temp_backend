# Temperaturüberwachungstool für Administratoren

<br>

## Installation
1. Github - Repo auf lokale Mashine Ziehen
2. Main.py ausführen

<br>

#### Alternativ
In der Kommandozeile den Pfad aufrufen, indem sich die [main.py](https://github.com/ibozorus/snif_temp_backend/blob/master_python/src/main.py) befindet. <br>
Dann folgenden Command eingeben
```
  python main.py
```
